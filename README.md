# 人生的关键窗口

一个以“年龄变化”为核心交互的人生节奏可视化网站。

用户可以通过拖动年龄滑块，观察不同人生事件在 `1-100 岁` 区间内的重要程度如何变化，并通过卡片和曲线图理解不同选择在不同年龄下的窗口感、优先级与节奏差异。

## 项目亮点

- 年龄滑块实时联动全站事件状态
- 事件卡片按当前年龄动态显示重要程度
- 支持按事件类别筛选
- 支持按“当前重要程度”排序
- 点击卡片可查看 `1-100 岁` 的趋势曲线
- 首页集成背景音乐播放器
- 纯静态前端实现，适合直接部署到 GitHub Pages

## 当前功能

网站当前包含以下能力：

- 顶部年龄控制区
- 当前年龄与人生阶段提示
- 事件总览与分类切换
- 超过 30 个细分人生事件
- 事件卡片重要度标签
- 事件详情弹窗与趋势图
- 默认尝试自动播放首页音乐

## 本地运行

这是一个纯静态项目，直接打开 `index.html` 即可运行。

如果你想用本地服务器方式访问，可以任选一种：

### 方式 1：VS Code Live Server

直接右键 `index.html`，选择 `Open with Live Server`。

### 方式 2：Python

```bash
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```

## GitHub Pages 部署

这个项目适合直接部署到 GitHub Pages。

### 步骤

1. 新建一个 GitHub 仓库
2. 把项目文件上传到仓库根目录
3. 进入仓库 `Settings -> Pages`
4. 在 `Build and deployment` 中选择：
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/(root)`
5. 保存后等待 GitHub Pages 发布

默认访问地址通常是：

```text
https://你的用户名.github.io/仓库名/
```

## 设计定位

这个网站是一个“人生节奏模拟工具”，不是标准答案生成器。

它更适合用来帮助用户理解：

- 同一件事在不同年龄下的重要程度为什么不同
- 人生选择为什么会受时间窗口影响
- 哪些事情适合现在做，哪些事情适合提前准备
